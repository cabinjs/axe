## Install

[npm][]:

```sh
npm install @ladjs/logger
```

[yarn][]:

```sh
yarn add @ladjs/logger
```


## Usage

```js
const Logger = require('@ladjs/logger');

const logger = new Logger();

console.log(logger.info('hello world'));
// info: hello world
```
